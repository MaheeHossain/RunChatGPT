1. Create repo and clone it
2. Create two folders - frontend and server
3. In frontend - npx create-react-app ./
4. In server - npm init -y, npm install body-parser cors express mongoose nodemon
5. Server package.json - 
    "type": "module",
    "scripts": {
        "start": "nodemon index.js"
    },
6. In frontend - npm install axios moment react-file-base64 redux redux-thunk
7. In frontend - npm install --legacy-peer-deps
